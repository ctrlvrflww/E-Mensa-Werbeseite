• div ist für große Abschnitte
  • span ist für kleine Textteile
  • haben keine semantische bedeutung
  • dienen zur Strukturierung und Styling von Inhalten mittels CSS

Span

  • ist ein inline Element
  • wird oft für kleinere textabschnitte verwendet, die individuell gestyled werden müssen, ohne den fluss des text zu unterbrechen

Div

  • ist ein Block-Level-Element
  • nimmt die gesamte Breite seines übergeordnetes Containers ein und beginnt immer auf einer neuen Zeile
  • wird verwendet, um gröere Abschnitte oder Container für Layout-Zwecke zu erstellen
